1. Introduction

Differential equations relate functions with their derivatives, forming the backbone of mathematical modeling in physics, engineering, biology, and economics.

Ordinary Differential Equations (ODEs) involve one independent variable, while Partial Differential Equations (PDEs) involve multiple.

2. First-Order ODEs

General linear form:

$$\frac{dy}{dx} + P(x)y = Q(x)$$

Solution using integrating factor $\mu(x) = e^{\int P(x)\,dx}$:

$$y \cdot \mu(x) = \int \mu(x) Q(x)\, dx + C$$

3. Second-Order Linear ODEs

Constant coefficients: $ay'' + by' + cy = 0$

RootsSolution
Real distinct$y = c_1 e^{r_1 x} + c_2 e^{r_2 x}$
Repeated$y = (c_1 + c_2 x)e^{rx}$
Complex $\alpha \pm \beta i$$y = e^{\alpha x}(c_1 \cos \beta x + c_2 \sin \beta x)$

4. Laplace Transforms

Converts ODEs → algebraic equations:

$$\mathcal{L}\{y''(t)\} = s^2 Y(s) - s y(0) - y'(0)$$

5. Systems of Differential Equations

Lotka-Volterra predator-prey model:

$$\frac{dx}{dt} = \alpha x - \beta xy, \quad \frac{dy}{dt} = \delta xy - \gamma y$$
